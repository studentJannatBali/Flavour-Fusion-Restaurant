# Flavour Fusion Restaurant Website

## Project Overview
**Flavour Fusion Restaurant** is a modern, responsive, single-page website designed to showcase the restaurant’s menu, chefs, and contact information. It provides an attractive, user-friendly interface for visitors to explore the restaurant’s offerings and get in touch.

---

## Features
- **Hero Section**: Displays the restaurant name, tagline, and a call-to-action button.  
- **Navigation Bar**: Smooth navigation to different sections: Home, Menu, Chef, About, Contact.  
- **Menu Section**: Divided into Breakfast, Lunch, and Dinner with images, dish names, and prices.  
- **Chef Section**: Showcases chefs with images and names.  
- **About Section**: Explains the restaurant’s mission and values.  
- **Contact Section**: Includes address, phone, email, social media links, and a contact form.  
- **Footer**: Social media icons and copyright information.  

---

## Technologies Used
- **HTML5** – Page structure  
- **CSS3** – Styling and layout  
- **Media Queries** – Responsive design for mobile and tablet screens  

---

## How to Run
1. Download or clone the repository.  
2. Open `index.html` in any modern web browser.  
3. Ensure images are in the same folder as `index.html` for proper display.
